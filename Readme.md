## Overview

This program demonstrates the use of variables, operators, and control flow statements in Python. It simulates a simple login system and calculates a student's final score and grade.

## Concepts Used
Variables to store usernames, passwords, and scores
Operators such as +, ==, and >= for calculations and comparisons
Logical operator and to check multiple conditions
Control flow using if, elif, and else to decide which code runs

# How the Program Works
The user enters a username and password.
If the login is correct, the user enters an exam score.
A bonus score is added to the exam score.
The program evaluates the final score and prints the grade.
